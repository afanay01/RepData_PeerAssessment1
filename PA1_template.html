Fill in all of the missing values in the dataset: Missing values are imputed as the mean of the specific 5-min interval. Create a new dataset that is equal to the original dataset but with the missing data filled in.

db.imp = db

impute.mean <- function(x) replace(x, is.na(x), mean(x, na.rm = TRUE))
db.imp = ddply(db.imp, ~ interval, transform, steps = impute.mean(steps))
